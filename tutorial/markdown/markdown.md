# Markdown 零基础入门教程

## 什么是 Markdown?

Markdown 是一种轻量级的标记语言,就像是给纯文本添加格式的"快捷键"。你只需要用一些简单的符号,就能让文字变得有层次、有重点。笔者就是Markdown重度使用用户，这里因为展开一些工作，需要做一下Markdown培训。

- 📝 简单易学,5分钟上手
- ⚡ 写作效率高,不用鼠标点来点去
- 🌐 广泛应用:博客、笔记、文档、GitHub等
- 💻 纯文本格式,任何设备都能打开

------

## 一、标题:文章的骨架

在文字前面加上 `#` 号,就能创建标题。`#` 越多,标题级别越小。

### 写法示例:

```markdown
# 一级标题(最大)
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题(最小)
```

### 显示效果:

# 一级标题(最大)

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题(最小)

**小贴士:** `#` 号后面要加一个空格哦!

## 二、文字样式:让文字有个性

### 2.1 加粗

用 `**文字**` 或 `__文字__` 把文字包起来

```markdown
这是**加粗的文字**
这也是__加粗的文字__
```

显示效果:这是**加粗的文字**,这也是__加粗的文字__

### 2.2 斜体

用 `*文字*` 或 `_文字_` 把文字包起来

```markdown
这是*斜体文字*
这也是_斜体文字_
```

显示效果:这是*斜体文字*,这也是_斜体文字_

### 2.3 加粗+斜体

用 `***文字***` 把文字包起来

```markdown
这是***加粗又斜体的文字***
```

显示效果:这是***加粗又斜体的文字***

### 2.4 删除线

用 `~~文字~~` 把文字包起来

```markdown
这是~~要删除的文字~~
原价~~299元~~,现价99元!
```

显示效果:这是~~要删除的文字~~,原价~~299元~~,现价99元!

## 三、列表:整理信息的好帮手

### 3.1 无序列表(项目符号)

用 `-`、`+` 或 `*` 开头,后面加空格

```markdown
- 苹果
- 香蕉
- 橙子

* 学习 Markdown
* 写文档
* 分享笔记

+ 任务一
+ 任务二
+ 任务三
```

显示效果:

- 苹果
- 香蕉
- 橙子

### 3.2 有序列表(数字编号)

用 `1.`、`2.`、`3.` 开头,后面加空格

```markdown
1. 打开电脑
2. 启动编辑器
3. 开始写作
4. 保存文档
```

显示效果:

1. 打开电脑
2. 启动编辑器
3. 开始写作
4. 保存文档

### 3.3 嵌套列表(多层级)

用缩进(Tab 或 2-4个空格)来创建子列表

```markdown
1. 水果
   - 苹果
   - 香蕉
2. 蔬菜
   - 白菜
   - 萝卜
     - 红萝卜
     - 白萝卜
```

显示效果:

1. 水果
   - 苹果
   - 香蕉
2. 蔬菜
   - 白菜
   - 萝卜
     - 红萝卜
     - 白萝卜

## 四、链接和图片

### 4.1 链接

格式: `[显示的文字](链接地址)`

```markdown
[百度](https://www.baidu.com)
[访问我的博客](https://myblog.com)
```

显示效果:[百度](https://www.baidu.com/)

### 4.2 图片

格式: `![图片描述](图片地址)`

```markdown
![可爱的猫咪](https://example.com/cat.jpg)
```

**小技巧:** 图片描述会在图片无法显示时出现

## 五、引用:引用别人的话

用 `>` 开头表示引用

```markdown
> 这是一句引用的话
> 可以连续多行

> 鲁迅说过:
> "我没说过这句话。"
```

显示效果:

> 这是一句引用的话 可以连续多行

> 鲁迅说过: "我没说过这句话。"

### 嵌套引用

```markdown
> 第一层引用
>> 第二层引用
>>> 第三层引用
```

显示效果:

> 第一层引用
>
> > 第二层引用
> >
> > > 第三层引用

## 十一、常用技巧总结

### 换行技巧

- 段落之间空一行
- 行尾加两个空格再回车(强制换行)
- 使用 `<br>` 标签

### 转义字符

如果要显示特殊符号本身,在前面加 `\`

```markdown
\* 这不是斜体
\# 这不是标题
```

### 快捷键记忆

- `#` = 井号 = 标题
- `*` = 星号 = 强调(加粗/斜体)
- `-` = 减号 = 列表
- `>` = 大于号 = 引用
- \``` = 反引号 = 代码
- `[]` = 方括号 = 链接/图片/任务
